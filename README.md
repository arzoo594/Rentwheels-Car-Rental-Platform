
**RentWheels** is a car rental service platform that allows users to:

- Browse available cars for rent.
- Book cars with flexible rental periods.
- Manage reservations and payments.
- Admin can manage cars, users, and bookings efficiently.

This repository contains the **server-side code** including API endpoints, database models, authentication, and middleware.
